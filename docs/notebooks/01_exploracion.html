1 Introducción

Este cuaderno explora los dos datasets proporcionados por Mexicanos Contra la Corrupción y la Impunidad (MCCI), que contienen registros de contratos de bienes y servicios celebrados por partidos políticos en México.

Nota

Los datos se encuentran en la carpeta datos/ del repositorio. No modifiques los archivos originales: cualquier transformación debe realizarse dentro de este notebook o en los scripts de la carpeta scripts/.

Los objetivos de este análisis son:

  1. Cargar y revisar la estructura de ambos datasets
  2. Identificar las diferencias entre contratos.csv y contratos_montos.csv
  3. Explorar la calidad de los datos (valores faltantes, rangos, distribuciones)
  4. Visualizar los datos más relevantes

  contratos        |> count(ano) |> mutate(dataset = "contratos.csv"),

  ggplot(aes(x = factor(ano), y = n, fill = dataset)) +

5.3 Monto total por partido

Ver código 
contratos_montos |>
  filter(costo > 0) |>
  group_by(partido) |>
  summarise(
    total_contratos = n(),
    monto_total     = sum(costo, na.rm = TRUE),
    .groups         = "drop"
  ) |>
  arrange(desc(monto_total)) |>
  mutate(partido = fct_reorder(partido, monto_total)) |>
  ggplot(aes(x = monto_total, y = partido)) +
  geom_col(aes(fill = partido), alpha = 0.85, show.legend = FALSE) +
  scale_x_continuous(labels = label_dollar(prefix = "$", suffix = " MXN",
                                           scale = 1e-6,
                                           suffix_after = FALSE,
                                           big.mark = ","),
                     name = "Monto total (millones de MXN)") +
  scale_fill_manual(values = paleta_partidos, na.value = "#9E9E9E") +
  labs(
    title    = "Monto total de contratos por partido político",
    subtitle = "Solo contratos con costo > 0 | contratos_montos.csv",
    y        = NULL
  ) +
  theme_minimal(base_size = 13) +
  theme(plot.title = element_text(face = "bold"))
